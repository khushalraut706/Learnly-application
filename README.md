# Learning Web

The Learnly project is a modern web application developed with HTML, CSS, and JavaScript, providing a user-friendly interface for online learning, course management, and interactive education features.

[Live Demo](https://khushalraut706.github.io/Learnly-application/)

## Features:

Interactive Learning: Watch educational content and access study materials through a clean and engaging interface.
Multiple Courses: Browse different categories such as Web Development, Programming, UI/UX Design, Data Science, and more.
Search & Filter: Easily search courses and filter content based on categories and topics.
Progress Tracking: Monitor learning progress and continue courses from where you left off.

## Tech Stack:
HTML: Used for structuring the web pages and creating user interface elements.
CSS: Employed for styling the platform and improving the overall user experience.
JavaScript: Implemented to handle course logic, search, filtering, enrollment, and dynamic interactions.
Bootstrap (optional): Utilized for responsive layouts and faster UI development with reusable components.
